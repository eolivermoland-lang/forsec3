Tjenester

Teknisk sikkerhet fra operatører med 16+ års erfaring fra norsk CERT/SOC-drift. Vi bygger, detekterer, responderer og herder.

01

Blue Team & Deteksjonsutvikling

Vi bygger deteksjonskapabiliteter som faktisk fanger reelle trusler. Fra Sigma- og YARA-regler til fullstendige detection-as-code-pipelines — vi designer, implementerer og tuner deteksjonslogikk tilpasset ditt miljø.

For SOC-team og virksomheter som vil bygge eller forbedre sine deteksjonsevner

02

Digital etterforskning og hendelsesrespons

Når hendelsen inntreffer, teller minuttene. Vi gjennomfører disk-, minne- og nettverksforensikk, sikrer bevis og leverer rapporter som holder i retten. Vi tilbyr også retainer-avtaler for garantert responstid.

For virksomheter som opplever eller vil forberede seg på sikkerhetshendelser

03

Sikkerhetstesting og sårbarhetsvurdering

Grundig teknisk testing av infrastruktur, applikasjoner og skyløsninger. Vi finner sårbarhetene før angripere gjør det — med konkrete funn og prioriterte anbefalinger.

For virksomheter som vil kjenne sine tekniske svakheter og lukke dem

Tjenestekatalog

Alle våre tjenester i detalj

Konkrete leveranser, tydelig scope, og hvem hver tjeneste passer for.

01

Deteksjonsutvikling

Strukturert utvikling og forvaltning av deteksjonsregler i din SIEM.

  • Sigma/YARA/KQL-deteksjonsregler
  • SIEM-tuning og optimalisering
  • detection-as-code pipeline
  • trusseljaktrapporter
02

Hendelsesrespons og forensikk

Rask, strukturert respons og digital etterforskning med bevissikret rapport.

  • Forensisk analyse (disk/minne/nettverk)
  • bevissinnhenting og beviskjede
  • malware-analyse
  • retainer-avtale med SLA
03

Penetrasjonstesting

Reell testing av angrepsflaten din mot infrastruktur, web, API og sky.

  • Sårbarhetsskanning (intern/ekstern)
  • penetrasjonstest av webapplikasjoner og APIer
  • Active Directory-sikkerhetsvurdering
  • sky-sikkerhetsgjennomgang (Azure/AWS/GCP)
04

Purple teaming

Samarbeid mellom angrep og forsvar i kontrollerte øvelser basert på MITRE ATT&CK.

  • MITRE ATT&CK-basert emulering
  • deteksjonsgapanalyse
  • angrepssimulering med sanntids blue team-tilbakemelding
  • måldrevet forbedringsplan
05

Herding — on-prem og sky

Systematisk herding av servere, nettverk, Active Directory og skyplattformer etter CIS-benchmarks.

  • Windows/Linux-herding (CIS-benchmarks)
  • Active Directory-herding og tiering
  • Azure/M365-sikkerhetskonfigurasjon
  • container- og Kubernetes-sikkerhet
06

Tabletop-øvelser og trening

Realistiske scenarier som tester organisasjonens evne til å håndtere sikkerhetshendelser.

  • Ransomware-scenario for ledelsen
  • teknisk IR-drill for sikkerhetsteam
  • krisehåndteringsøvelse
  • evalueringsrapport med forbedringstiltak
